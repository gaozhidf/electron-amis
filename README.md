## electron amis demo

- electron + amis
- init from: https://github.com/electron-react-boilerplate/electron-react-boilerplate.git
- amis: https://github.com/baidu/amis
- amis config file: `src/main/renderer/components/amisConfig.json`

#### command

```bash
# 在国内使用 cnpm install

npm install
npm start
npm run build
```
